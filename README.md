# FlowUs Pro V13 - 模块化版

一个功能强大的本地优先笔记应用，采用模块化架构设计，支持多种高级视图和本地文件存储。

## 功能特性

- 📝 类似 Notion 的块状编辑器
- 📋 可编辑的垂直时间轴
- 📅 可拖拽的横向甘特图（支持缩放）
- 📊 看板视图
- 📈 智能表格
- 📁 子页面支持
- 💾 原生文件系统 API 支持（.json 文件格式）
- 🌙 深色模式支持
- 📱 响应式设计

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+ 模块化)
- 原生文件系统 API

## 核心功能详解

### 编辑器
- 支持文本、标题、待办事项等多种块类型
- 通过 "+" 按钮或 "/" 命令插入新块
- 智能 slash 菜单
- 拖拽重新排序内容块

### 垂直时间轴
- 可编辑的时间节点
- 支持修改标题和日期

### 横向甘特图
- 完全可交互的拖拽功能
- 支持移动事件条和调整大小
- 多种缩放级别（年/月/周/日）
- 全屏模式查看
- 事件状态管理（进行中/已完成/未开始/阻塞/里程碑）

### 看板视图
- 可自定义列
- 卡片拖拽移动
- 支持添加新列和卡片

### 表格
- 可编辑的智能表格
- 支持动态添加行

### 子页面
- 支持创建层级页面结构
- 页面间跳转

## 使用方法

直接在支持原生文件系统 API 的现代浏览器（如 Chrome）中打开 `index.html` 文件即可使用。

### 新建项目
1. 点击"新建本地项目"
2. 在弹出的对话框中选择保存位置并命名文件（.json 格式）
3. 开始使用

### 打开项目
1. 点击"打开 .json 文件"
2. 选择之前保存的项目文件
